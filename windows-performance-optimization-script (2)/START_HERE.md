## Step 3: Configure for Free Fire

### Quick Setup (30 seconds):

**A. Choose Profile:**
- At the top, click **"Gaming"** button
- This enables recommended settings

**B. Enable AI Features:**
- Scroll to "Gaming Performance" section
- Click the switch for **"AI Drag Smoothing"** → Turns ON (gold)
- Click the switch for **"Free Fire Mode"** → Turns ON (gold)

**C. Adjust Settings:**
- Below AI Drag Smoothing, you'll see sliders
- Drag **"Smoothing Intensity"** to **70%**
- Drag **"Prediction Strength"** to **45%**

**D. Activate:**
- Click the big gold button **"▶ Activate"** at the top
- It turns RED and says "⏹ Deactivate"
- Status changes to "OPTIMIZED"

**E. Done!**
- You're all set!
- Start playing Free Fire!

## 🎛️ Using the Controls

### Toggle Switches:
```
[● ────── OFF]  ← OFF position (gray)
[──── ON ●]     ← ON position (gold)
```

**How to use:**
- Click the switch to toggle
- Slides from left (OFF) to right (ON)
- Gold = ON, Gray = OFF

### Sliders:
```
Smoothing Intensity         [70%]
[━━━━━━━●━━━━━]
Raw ←→ Ultra Smooth
```

**How to use:**
- Click and drag the circle
- Or click on the bar to jump to position
- Value shows in the gold badge

### Buttons:
```
[▶ Activate]     ← Gold = ready to activate
[⏹ Deactivate]  ← Red = currently active
```

**How to use:**
- Just click to toggle
- Gold → Red when activating
- Red → Gold when deactivating

---

## 🔒 Important: Settings Lock

**When you click Activate:**
- All settings become **locked** (grayed out)
- You can't change anything while active
- This prevents conflicts

**To change settings:**
1. Click **"⏹ Deactivate"**
2. Make your changes
3. Click **"▶ Activate"** again

## 📊 What Do The Numbers Mean?

### Performance Score (Right Sidebar):
- **80-100** 🟢 = Optimal (best settings)

## 🆘 Troubleshooting

### "The file won't open"
- Try a different browser (Chrome recommended)
- Make sure you're opening `dist/index.html` not just `index.html`
- Right-click → Open with → Chrome

### "I don't see the app"
- Check if popup blocker blocked it
- Try refreshing the page (F5)
- Make sure JavaScript is enabled

### "Buttons don't work"
- Check if system is active (settings might be locked)
- Try clicking Deactivate first
- Refresh the page and try again

### "Nothing happens when I click Activate"
- This is a demonstration interface
- It shows how the tool works
- To actually optimize your PC, you'd need native software
- This version is for configuring and visualizing settings

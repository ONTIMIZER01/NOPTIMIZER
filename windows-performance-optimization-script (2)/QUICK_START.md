# 🚀 Quick Start Guide

## 30 Second Setup for Free Fire

1. **Open Opnonti Pro**
2. **Click "Gaming" profile** (top right)
3. **Enable these features:**
   - ✅ Disable Mouse Acceleration
   - ✅ AI Drag Smoothing
   - ✅ Free Fire Mode
   - ✅ Input Lag Reduction
   - ✅ Polling Rate Boost

4. **Configure AI settings:**
   - Smoothing Intensity: **70%**
   - Prediction Strength: **45%**

5. **Click "▶ Activate"**
6. **Start Free Fire!**

---

## Understanding the Interface

### Header Section
- **Profile Buttons** - Quick switch between Gaming/General/Custom
- **System Status** - Shows if optimizations are active
- **Activate Button** - Master on/off switch (gold when ready, red when active)

### Main Panel (Left)
Three sections of features:
1. **Mouse & Pointer** (Blue icon) - Basic mouse settings
2. **Gaming Performance** (Red icon) - AI and gaming features
3. **System Performance** (Green icon) - Low-level optimizations

### Sidebar (Right)
- **Live Monitoring** - Real-time performance score and metrics
- **Quick Actions** - Export config, reset, view logs
- **Profile Info** - Current configuration details
- **Help & Tips** - Quick tips

---

## Feature Toggle Switches

### How They Work
- **Left (OFF)** - Feature is disabled (gray)
- **Right (ON)** - Feature is enabled (gold)
- **Locked** - Can't change when system is active

### Visual Indicators
- **Icon Background** - Lights up gold when active
- **Active Badge** - Shows when feature is running
- **Color Coding** - Green = Recommended, Purple = Advanced, Blue = AI, Red = Gaming

## AI Drag Smoothing Settings

### Smoothing Intensity Slider
```
0%     Raw input, no smoothing
25%    Slight smoothing, mostly raw
50%    Balanced smoothing
75%    Heavy smoothing (recommended for Free Fire)
100%   Maximum smoothing, ultra stable
```

### Prediction Strength Slider
```
0%     No prediction
25%    Light prediction
50%    Moderate prediction (recommended for Free Fire)
```

## Understanding the Performance Score

Your performance score (shown in sidebar) is calculated based on:

- **50** - Base score
- **+10** - Input Lag Reduction enabled
- **+10** - Polling Rate Boost enabled
- **+8** - Response Optimization enabled
- **+7** - Hardware Acceleration enabled
- **+5** - Enhanced Pointer Precision enabled
- **+10** - AI Drag Smoothing enabled

### Score Ranges
- **80-100** 🟢 Optimal - Maximum performance
- **60-79** 🟡 Good - Decent performance
- **0-59** 🔴 Basic - Room for improvement

## System Metrics Explained

### Input Latency
- **~8ms** - Standard Windows latency
- **~1ms** - Optimized latency (with Input Lag Reduction)

### Polling Rate
- **125Hz** - Standard USB polling (8ms updates)
- **1000Hz** - Maximum polling (1ms updates)

### CPU Usage
- **1-2%** - Inactive
- **2-4%** - Active with optimizations

### GPU Acceleration
- **Disabled** - CPU handles cursor rendering
- **Enabled** - GPU handles cursor rendering (smoother)

## Quick Tips

### ✅ DO
- Start with Gaming profile for best defaults
- Test settings in practice mode first
- Deactivate before changing settings
- Export your config when you find perfect settings
- Monitor the performance score

### ❌ DON'T
- Change settings while system is active (they're locked anyway)
- Use 100% smoothing for competitive flick shots
- Enable ALL features at once (some conflict)
- Forget to activate after configuring
- Use Free Fire Mode for other games

## Troubleshooting

### Settings won't change
**Solution:** Click "⏹ Deactivate" first

### Mouse feels weird
**Solution:** Start with lower smoothing (40-50%), gradually increase

### Input lag feels high
**Solution:** Enable "Input Lag Reduction" and "Polling Rate Boost"

### Aim is too smooth/floaty
**Solution:** Reduce smoothing intensity or disable AI Drag Smoothing

### Aim is too shaky
**Solution:** Enable "Micro-Movement Filter" and increase smoothing
